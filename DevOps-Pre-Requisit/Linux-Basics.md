# Basic Commands

## Print to screen
````bash
echo Hi  
 ````
 Output
 ````bash
Hi  
 ````
## List Files & Folders
````bash
    ls
````
 Output
 ````bash
 File.txt, my_dir1, file2.conf
````
## Change directory
````bash
cd my_dir1
````
## Present Working Directory
````bash
pwd
````
Output
````bash
/home/my_dir1
````
## Make Directory
````bash
mkdir new_directory
````
## Multiple commands
````bash
cd new_directory; mkdir www; pwd
````
Output
````bash
/home/my_dir1/new_directory
````
