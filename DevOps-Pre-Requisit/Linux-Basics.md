List Files & Folders

    ls -------> File.txt, my_dir1, file2.conf

Change directory

    cd my_dir1

Present Working Directory

    pwd -------> /home/my_dir1

Make Directory

    mkdir new_directory

Multiple commands

    cd new_directory; mkdir www; pwd   --------> /home/my_dir1/new_directory
